# Computer Science Examples

Examples that demonstrate computational concepts, algorithms, distributed systems, or coordination problems using agent-based modeling.

---

### Conway's Game of Life (Fast)
An optimized implementation of Conway's Game of Life using Mesa's PropertyLayer abstraction and convolution-based neighbor counting to achieve significantly improved simulation performance.

**Example:**  
[examples/conways_game_of_life_fast](../examples/conways_game_of_life_fast)

---

### Dining Philosophers
An implementation of the classic Dining Philosophers coordination problem demonstrating resource contention, deadlock, and fairness strategies in distributed systems.

**Example:**  
[examples/dining_philosophers](../examples/dining_philosophers)

---

### Conway's Game of Life on a Hexagonal Grid
A hexagonal-grid variation of Conway's Game of Life exploring cellular automaton dynamics and pattern formation under alternative neighborhood rules.

**Example:**  
[examples/hex_snowflake](../examples/hex_snowflake)

---

### Shape Model – Basic Grid with Two Agents
A minimal example demonstrating Mesa's web-based grid visualization, showcasing custom agent shapes, directional arrowheads, and text overlays in the browser.

**Example:**  
[examples/shape_example](../examples/shape_example)

---

### Pseudo-Warehouse Model (Meta-Agent Example)
A demonstration of Mesa's meta-agent capability where robot agents are composed of coordinated sub-agents, illustrating hierarchical and multi-level agent architectures.

**Example:**  
[examples/warehouse](../examples/warehouse)
